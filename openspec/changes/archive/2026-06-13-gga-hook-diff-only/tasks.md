- **Files**: `scripts/gga-pre-commit.mjs` (create, ~120 LOC)
- **Body**: ESM module per design skeleton. Node stdlib only (`child_process`, `fs`, `path`). Implements the 6-step data flow (read .gga-ignore → parse diff → spawn `gga run` → parse output → filter → report). **Exports `filterFindings(findings, changedLines, ignoreSet, killSwitch)` for unit testing in T6** (T6 was written against this export — apply T7 BEFORE T6 in the wrong direction would fail).   - Empty diff (`git commit --allow-empty`) → exits 0 silently
  - `GGA_DIFF_FILTER=off` → pass-through, exits with GGA's code
  - `GGA_DIFF_FILTER=off` + parse failure → exits with GGA's code (NOT 0)
  - `parseGgaIgnore` syntax error → fails CLOSED (exit 1 with clear message)
  - `chmod +x scripts/gga-pre-commit.mjs`
